# Projet WS - Gestion des Étudiants

## Description
Application mobile Android pour la gestion des étudiants avec un web service PHP et une base de données MySQL.

https://github.com/user-attachments/assets/82cc1ca9-0652-4af8-bdac-79ac7b54ab9e


## Technologies utilisées

### Backend
- **PHP 8** : Web service RESTful
- **MySQL** : Base de données
- **XAMPP** : Serveur local

### Frontend (Android)
- **Java** : Langage de développement
- **Volley** : Requêtes HTTP
- **Gson** : Parsing JSON
- **Android Studio** : IDE de développement
